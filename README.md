# Data Structures Crammer

A fast, kid-friendly way to revise Data Structures before your exam!

## How to use:
To run this project locally or on GitHub Pages:
1. Push this whole folder to a GitHub repository.
2. Go to your repository settings > Pages > Source > select "main" branch.
3. Your site will be live!

The site consists of purely static HTML, CSS, JS, and JSON files, so it can also be opened directly by double-clicking `index.html` in your browser.

## Converting the Markdown to PDF
1. Included is a file named `crash_guide_by_bahnasy.md`.
2. Open it in VSCode (with a Markdown PDF extension), or use an online Markdown-to-PDF converter (like markdowntopdf.com).
3. The styling will be automatically applied based on simple markdown elements. You can also print the "Study Guide" section directly from the web app using `Ctrl+P` (Save to PDF).
